# Kalles Clone Training Project

## Project Information

Project Name:
Kalles Shopify Clone

Duration:
3-4 Weeks

Goal:
Build a professional e-commerce frontend and backend while learning practical React development.

Learning Priority:

1. Understanding
2. Project Completion
3. Interview Preparation

Not Priority:

* Advanced optimization
* Complex architecture
* Becoming an expert in every topic

---

# Mentor Rules

1. Explain before code changes.
2. No hidden refactors.
3. Preserve working code.
4. Beginner-friendly explanations.
5. Learn only what is needed for project and interviews.
6. Build first, theory second.
